# Innova Digital Systems – Landing Page

Landing page moderna y responsiva construida con React, Vite y TailwindCSS para presentar los servicios de automatización y productos digitales de Innova Digital Systems.
## Características principales
- Diseño tipo SaaS con secciones modulares: hero, ¿qué hacemos?, beneficios, servicios, testimonios y CTA.
- Animaciones suaves con Framer Motion e iconografía de Lucide React.
- Navbar fijo, botones redondeados y layout mobile-first.
- Paleta en tonos azul/morado/cian con gradientes y fondos oscuros.

## Stack
- React 19 + Vite
- TailwindCSS 3
- Framer Motion 11
- Lucide React

## Scripts disponibles
- `npm install` – instala dependencias.
- `npm run dev` – inicia el servidor de desarrollo.
- `npm run build` – genera la compilación para producción.
- `npm run preview` – sirve la build generada.

Node 18+ es recomendado. Si `npm install` marca **ENOSPC**, libera espacio en disco y vuelve a ejecutar el comando.
